# 선물하기 도메인 용어집

선물하기 팀 내부에서 쓰는 축약어·고유명사. 모르는 용어를 만나면 여기서 확인하고, 없으면 사용자에게 의미를 물어본 뒤 확정된 내용을 추가한다.

## O2

CSS 마크업 리포지토리. `github.daumkakao.com/kakao-gift/O2`. 마크업은 외주사에서 작업하고, 선물하기 프로젝트는 O2 브랜치를 당겨받아 사용한다.

## fetch markup

O2 리포의 마크업 브랜치를 선물하기 프로젝트 쪽으로 당겨오는 작업.
